			==========================================================================
						Base Conversion Functions in Python
			=========================================================================
=>The Process of Converting One Number System Value into another Number System Value is Called Base Conversion.
=>In Python Programming, we have 3 Base Conversion Functions. They are

				1. bin()
				2. oct()
				3. hex()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1. bin()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:      varname=bin(Decimal / octal / HexaDecimal Value)
=>This Function is used Converting any Number System Value into Binary.
--------------------------
Examples
--------------------------
>>> a=15
>>> x=bin(a)
>>> print(x)------------------------------0b1111
>>> b=20
>>> x=bin(b)
>>> print(x)----------------0b10100
----------------------------
>>> a=0o15
>>> b=bin(a)
>>> print(b)----------0b1101
-------------------------------
>>> a=0xF
>>> b=bin(a)
>>> print(b)--------------0b1111
>>> a=0xBEE
>>> b=bin(a)
>>> print(b)--------------0b101111101110
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
2. oct()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:      varname=oct(Decimal / Binary / HexaDecimal Value)
=>This Function is used Converting any Number System Value into Octal.
-------------------------
Examples
-------------------------
>>> a=18
>>> b=oct(a)
>>> print(b)------------0o22
--------------------------
>>> a=0b1111
>>> b=oct(a)
>>> print(b)------------0o17
-----------------------------
>>> a=0xF
>>> b=oct(a)
>>> print(b)------------0o17
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
3. hex()
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
=>Syntax:      varname=hex(Decimal / Binary / Octal Value)
=>This Function is used Converting any Number System Value into Hexa Decimal Value.
-----------------------------
Examples
-----------------------------
>>> a=3054
>>> b=hex(a)
>>> print(b)-------------------0xbee
>>> a=172
>>> b=hex(a)
>>> print(b)------------------0xac
>>> a=15
>>> b=hex(a)
>>> print(b)------------------0xf
>>> a=0xa
>>> b=hex(a)
>>> print(b)--------------0xa
----------------------------------
>>> a=0b1111
>>> b=hex(a)
>>> print(b)---------------0xf
>>> a=0b1010
>>> b=hex(a)
>>> print(b)-------------0xa
--------------------------------------------
>>> a=0o17
>>> b=hex(a)
>>> print(b)----------------0xf
>>> a=0o123
>>> b=hex(a)
>>> print(b)---------------0x53
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
